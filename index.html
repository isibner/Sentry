// Sentry

Meet Sentry.

Wherever your code is hosted, Sentry is ever vigilant.

He'll tirelessly watch your source to help you track down bad practices.

Get Sentry

Fully Extensible
With Customisable Services

Sentry watches your code for changes and passes them along to services, which can be configured for fine-grained control over how they're applied. You can also add custom services for any use case.

Use Any Git Source
Track Anything, Anywhere

Sentry can pull code from many different sources using sources. Github works right out of the box! If there's not a source for your host of choice, it's easy to write your own.
